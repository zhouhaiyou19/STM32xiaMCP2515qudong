# STM32下MCP2515驱动

## 简介
本仓库提供了一个基于STM32的MCP2515物理层驱动代码。该代码仅用于研究和学习目的，是从相关项目工程中提取出来的，主要测试了扩展帧的功能。代码中默认波特率为50K，使用了SPI2 + DMA进行通信，未使用MCP2515的中断引脚。

## 硬件连接
- CS引脚：PB12
- MISO引脚：PB14
- SCK引脚：PB13
- MOSI引脚：PB15
